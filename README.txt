# Installation Guide

Welcome to the TyperZ installation guide.

To install the software, please follow these steps:

1. **Download the Setup File:**
   - Ensure you have the setup file ('setup.exe`) located on your computer or inside TyperZ package.

2. **Run the Setup File:**
   - Double-click on `setup.exe` to start the installation process.

3. **Follow the On-Screen Instructions:**
   - The setup wizard will guide you through the installation steps.
   - Click "Next" to proceed through the installation screens.
   - Choose your preferred installation options when prompted.

4. **Complete the Installation:**
   - Once the installation is complete, click "Finish" to exit the setup wizard.
   - You may need to restart your computer to finalize the installation.

5. **Launch the Application:**
   - Find and open the application from your Start menu or desktop shortcut.

Thank you for installing TyperZ!
